CSS Snippet CheatSheet

A handy-dandy collection of CSS Snippets for you to refer to whenever you get stuck!

Flexbox Row

Use these properties to create a Flexbox Row layout.

                    
    .row {
        display: flex;
        flex-direction: row;
        flex-wrap: wrap;
    }

Flexbox Column

Use these properties to create a Flexbox Column layout.

                    
    .column {
        display: flex;
        flex-direction: column;
    }

CSS Grid Layout

Build a 12-column layout by using a CSS grid.

                    
    .grid {
        display: grid;
        width: 100%;
        grid-template-columns:
    repeat (12, 1fr);
    }

Linear Gradients

This will create a background linear gradient from top to bottom.

                    
    .linear-gradient-background {
        background-image: linear-gradient(
            rgba(232, 102, 236, 0.3) 0%
            rgba(232, 102,236, 0.6) 100%
        );
    }

Box Transition Glow

Use transition and box shadows to glow on hover.

                    
    .code-card .card header {
        border-radius: 8px;
        transition: all 0.5s ease-in-ease out;
    }    
    .code-card:hover,
    .code card: hover .card-header {
        box-shadow: inset 0px 0px 8px
    rgba(232, 102, 236, 1), 0 0 15px
    rgba(232, 102, 236, 1);
    }

Overlay Card with Title

Use position properties and negative margins to raise elements higher than their natural starting point. 

                    
    .card header {
        position: relative;
        margin-top: -20px;
     
    }
